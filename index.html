Basic 3D Animation using Blender

Introduction

This is a 8 weeks course, and we have added some primary content in the preparatory week, to get things started. Since animation is an art which requires a tool to express the creativity, we have chosen Blender which is a free and open source software.

In this preparatory week, we discuss about the 3D visualization concept, have a small activity, talk about the system configuration required for using Blender, and explain the installation procedure. We expect that by going through this preparatory week, you would be ready to go in the course, with Blender software installed!

Mini Project 01

You will have to create a 3D model. This model should be based on a topic/concept from school level mathematics or science, where you feel 3D visualization will be effective. Create the necessary model, apply appropriate textures, materials, lights, and render the image. Example of such a model could be molecular structure of water, benzene, structure of our solar system, sundial, various laboratory apparatus etc.

All the best!

Mini Project

Page 60

Mini Project 02 - Faulty Car animation

The mini project is an animation activity, which requires you to apply the animation principles you have learnt so far. The topic for this animation is: A faulty car. Imagine a faulty car on a road, where it is not able to start off smoothly, goes haywire, breaks down (even literally!), loses some parts, etc. You can think of any such situation. We have divided this project into three stages. Please go through these suggestions, before you start:

Stage 1 - Planning:

  • Plan the animation for the entire scene
  • Keep the duration less than 30 seconds, so that rendering the scene does not take too much of machine time.
  • Time the animation properly, and get it reviewed form peers.

Stage 2 - Animation:

  • Create a low poly car model. Simpler the better is the mantra here! You will have less problems while animating it.
  • Create a suitable road/terrain to match the animation. Avoid unnecessary objects to make the motion visible.
